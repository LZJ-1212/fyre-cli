How to Play

  • Use your arrow keys (Up, Down, Left, Right) to move all tiles.
  • On touch devices, you can swipe in any direction.
  • When two tiles with the same number touch, they merge into one with their sum!
  • After every move, a new tile (value 2 or 4) will appear in a random empty spot.
  • The game is won when you create a tile with the number 2048.
  • The game ends when no more moves are possible.
  • Use the Save button to store your current game state and Load to resume it later.
